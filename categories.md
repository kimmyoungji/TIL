## React Dom
### tags
- ```<Router>``` : Router 가장 바깥 태그
- ```<Route path="패스"><Route/>``` : 주소 별 요소들을 묶어주는 태그
- ```<Link />``` : 기본 주소에 주소를 추가한 <a> 요소를 만들어준다
- ```<reDirect />``` : 한 주소에서 다른 주소로 변경해준다.
- ```<switch> <switch/>```: 주소 별로 다른 요소가 나타날 수 있게 해주는 태그
