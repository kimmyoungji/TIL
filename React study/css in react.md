# css in react 

## 4가지 css 적용 방식
1. inline
   * react 에서는 inline 보다, class명을 활용하는 것이 더 좋다.
   * 요소의 style 속성을 활용하여, inline css를 작성해 볼 수 있다. JS 오브젝트로 한번에 정의하여 style에 대입해주어도 된다.
2. js object
3. css file
   * 외부 css 파일을 import 하여 사용한다.
4. css modules
   * 요소의 class를 변경하여, 상황에 따라, 다른 css가 적용 되도록 만들 수 있다.
